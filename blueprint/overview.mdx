## Terms

- **Blueprint** — A collection of sheets that describe a complete dataset
- **Sheet** — A collection of fields that describe a single entity (eg. a person, a product, a company).
- **Field** — Describes an attribute of the entity (eg. name, email) and how it should behave.


<RequestExample>
```json Blueprint Example (simple)
{
  "sheets": [
    {
      "key": "products",
      "label": "Products",
      "description": "A list of products available for sale",
      "fields": [
        {
          "key": "code",
          "label": "Product Code",
          "type": "string"
        },
        {
          "key": "description",
          "type": "string"
        },
        {
          "key": "price",
          "type": "number"
        }
      ]
    }
  ]
}
```
</RequestExample>
<RequestExample>
In this example each Employee is recording their choice of Swag stored across 2 sheets
```json Blueprint Example (complex)
{
  "sheets": [
    {
      "key": "Employees",
      "label": "Employees",
      "description": "each employee in the company",
      "fields": [
        {
          "key": "eid",
          "label": "Employee ID string",
          "type": "string",
          "config": {"size": "tiny"},
          "constraints": [
            {
              "type": "unique",
              "config": {"case_sensitive": false}
            }
          ]
        },
        {
          "key": "first_name",
          "label": "First Name",
          "type": "string",
          "config": {"size": "tiny"},
          "constraints": [ {"type": "required"} ]
        },
        {
          "key": "last_name",
          "label": "Last Name",
          "type": "string",
          "config": {"size": "tiny"},
          "constraints": [ {"type": "required"} ]
        },
        {
          "key": "salary",
          "type": "number",
          "config": {"decimal_places": 2},
          "read-only": true
        },
        {
          "key": "department",
          "type": "enum",
          "config": {
            "options": [
              {"value": "Sales"},
              {"value": "Marketing"},
              {"value": "People Ops"}
            ],
            "allow_custom": false
          }
        },
        {
          "key": "swag_choice",
          "type": "reference",
          "config": {
            "ref": "company_swag",
            "relationship": "has-one"
          }
        },
        {
          "key": "swag_size",
          "type": "number",
          "config": {"decimal_places": 0}
        }
      ],
      "access": ["import", "edit"]
    },
    {
      "key": "company_swag",
      "label": "Swag Options",
      "fields": [
        {
          "key": "name",
          "label": "Product Code",
          "type": "string"
        },
        {
          "key": "date_available",
          "type": "date"
        }
      ]
      "access": []
    }
  ]
}
```
</RequestExample>
