Embed Javascript is a zero dependency front-end SDK wrapper around the Flatfile API. It allows you to securely and easily add an embedded data import experience, also known as a Space, to any Javascript app.

Using an iframe, Embed Javascript seamlessly integrates into your website or application, providing a simple yet robust experience for collecting and transforming data from your customers.
