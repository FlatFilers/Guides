title: "@flatfile/plugin-record-hook"
description:
  A plugin that exports a function named `recordHook` that sets up an event
  listener for the `commit:created` event. The purpose of this plugin is to
  provide a concise syntax for running custom logic on individual data records
  in Flatfile.
