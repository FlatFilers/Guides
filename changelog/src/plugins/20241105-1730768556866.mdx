### November 05, 2024
## schema

@flatfile/plugin-convert-yaml-schema@0.4.1
The @flatfile/plugin-convert-yaml-schema package has been updated to version 0.4.1. The update includes a dependency update for @flatfile/plugin-convert-json-schema to version 0.6.0. Additionally, there are changes to the structure of a column in the generated blueprint. The arrayColumn now has a type of string-list instead of an enum, and its description has been updated to "A column for string arrays!". The config property with options has been removed from this column. The objectColumn_nestedUniqueNumberColumn column has also been modified, but the details of the change are not provided.
