### November 07, 2024
## schema

@flatfile/plugin-convert-yaml-schema@0.5.0
The plugin @flatfile/plugin-convert-yaml-schema has been updated to version 0.5.0. The package bundler has been swapped to tsup. The minimum required Node.js version has been increased to 18. The testing setup has been migrated from Jest to Vitest. The dependencies @flatfile/plugin-space-configure and @flatfile/plugin-convert-json-schema have been updated to their latest versions 0.8.0 and 0.7.0 respectively. The NPM scripts have been updated to use the Vitest test runner. The Jest configuration file has been removed.
