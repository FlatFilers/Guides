### December 19, 2024
## core_plugin

@flatfile/plugin-space-configure@0.8.1
This release note updates the @flatfile/plugin-space-configure package to version 0.8.1. It adds a type definition for the tick() function used in the configureSpace() method. The tick() function is now typed as TickFunction imported from the @flatfile/plugin-job-handler package. This provides better type safety for consumers of the package when using the tick() function callback. Additionally, the @flatfile/plugin-job-handler dependency has been updated to version 0.8.1.

Example usage:

```typescript
import { configureSpace } from '@flatfile/plugin-space-configure';

configureSpace({
  // ...
}, async (event, workbookIds, tick) => {
  // tick is now typed as TickFunction
  await tick(0.5, 'Processing workbooks...');
  // ...
});
```
