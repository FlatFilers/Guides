### November 07, 2024
## extractors

@flatfile/plugin-job-handler@0.8.0
Updated the package bundler to tsup.
Incremented the package version from 0.7.0 to 0.8.0.
Updated the minimum required Node.js version to 18.
Replaced Jest with Vitest for running tests. The test script commands have been updated accordingly: "test" runs all tests, "test:unit" runs unit tests excluding end-to-end tests, and "test:e2e" runs end-to-end tests with no file parallelism.
Updated the @flatfile/util-common dependency to 1.6.0.
Updated the @flatfile/bundler-config-tsup and @flatfile/utils-testing dev dependencies to their latest versions.
Added a new dev dependency @flatfile/config-vitest for configuring Vitest.
Removed the Jest configuration file jest.config.cjs.
Added a new Vitest configuration file vitest.config.ts that imports the Vitest configuration from @flatfile/config-vitest.
Updated the job.handler.e2e.spec.ts file to use Vitest testing utilities and mocking utilities from vitest. The tests now use the TestListener and setupDriver utilities from @flatfile/utils-testing.
