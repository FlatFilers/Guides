### November 07, 2024
## automations

@flatfile/plugin-export-workbook@3.0.0
The release includes the following updates to the export-workbook plugin:

The package's bundler has been swapped to tsup. The minimum required Node.js version has been updated to >=18. The dependencies @flatfile/plugin-job-handler and @flatfile/util-common have been updated to the latest versions 0.8.0 and 1.6.0 respectively. The Jest testing framework has been replaced with Vitest, and the test script commands have been updated accordingly. For example, to run unit tests, use the command `npm run test:unit`, and for end-to-end tests, use `npm run test:e2e`. The dev dependency @flatfile/bundler-config-tsup has been updated to version 0.2.0, and a new dev dependency @flatfile/config-vitest has been added for configuring Vitest.
