## transform

@flatfile/plugin-dedupe@1.5.1
This release updates the @flatfile/plugin-dedupe package to version 1.5.1. The update adds a type for the tick() function parameter in the dedupe.plugin.ts file. Additionally, it updates the dependency @flatfile/plugin-job-handler to version 0.8.1. Developers using this package should be aware of the new type for the tick() function, which may require updating their code to conform to the new type. The update also includes changes to the package.json file to reflect the new version and dependency update.
