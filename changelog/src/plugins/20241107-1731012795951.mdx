### November 07, 2024
## core_plugin

@flatfile/plugin-space-configure@0.8.0
The release notes include the following updates:

The package has swapped the bundler to tsup. This change updates the build scripts and removes the jest configuration file.

The minimum required Node.js version has been updated to >= 18.

The testing setup has been migrated to use Vitest instead of Jest. New scripts have been added for running Vitest tests: `test`, `test:unit`, and `test:e2e`.

Dependencies have been updated:
- `@flatfile/plugin-job-handler` has been updated to version 0.8.0.
- `@flatfile/bundler-config-tsup` has been updated to version 0.2.0.
- `@flatfile/config-vitest` and `@flatfile/utils-testing` at versions 0.0.0 and 0.5.0 respectively have been added as dev dependencies.

The package version has been updated to 0.8.0.
