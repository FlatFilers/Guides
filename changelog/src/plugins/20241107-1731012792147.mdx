### November 07, 2024
## schema

@flatfile/plugin-convert-json-schema@0.7.0
The release updates the @flatfile/plugin-convert-json-schema package to version 0.7.0. The package now uses tsup as the bundler instead of the previous bundler. The minimum required Node.js version has been updated to >= 18. The jest testing setup has been replaced with vitest. The @flatfile/plugin-space-configure dependency has been updated to version 0.8.0. No new public interfaces have been introduced in this release. For developers using this package, they will need to ensure they are running Node.js version 18 or higher and update any testing setups that relied on jest.
