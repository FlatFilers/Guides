### November 07, 2024
## extractors

@flatfile/plugin-pdf-extractor@0.5.0
The release notes include the following updates:

The package now uses the tsup bundler instead of the previous bundler. This change swaps the bundler used for building the package.

The minimum required Node.js version has been updated to Node.js 18 or newer versions.

The dependencies @flatfile/util-file-buffer and @flatfile/util-common have been updated to their latest versions 0.6.0 and 1.6.0 respectively.

The testing setup has been updated to use Vitest instead of Jest. New scripts have been added for running unit tests and end-to-end tests separately with Vitest.

The devDependencies @flatfile/bundler-config-tsup and @flatfile/config-vitest have been updated to their latest versions.
