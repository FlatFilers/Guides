### November 07, 2024
## automations

@flatfile/plugin-automap@0.8.0
The release includes the following updates:

The package now uses tsup for bundling. The node engine requirement has been updated to version 18 or newer. The test suite has been migrated from Jest to Vitest. New scripts have been added for running Vitest tests: `test`, `test:unit`, and `test:e2e`. The peer dependency `@flatfile/util-common` has been updated to version 1.6.0. The end-to-end tests now use a TestListener and a WebDriver setup, improving the testing infrastructure. Overall, the release focuses on updating the bundler, test runner, and dependencies while maintaining the core functionality of the automap plugin.
