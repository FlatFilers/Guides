### November 07, 2024
## schema

@flatfile/plugin-convert-sql-ddl@0.4.0
The package @flatfile/plugin-convert-sql-ddl has been updated to version 0.4.0. The bundler has been swapped from the previous bundler to tsup. The package now requires Node.js version 18 or higher. The dependencies @flatfile/plugin-space-configure and @flatfile/plugin-convert-json-schema have been updated to their latest versions 0.8.0 and 0.7.0 respectively. The testing setup has been migrated from Jest to Vitest, and separate commands have been added for running unit tests and end-to-end tests. The package.json has been updated to reflect these changes in the scripts section.
