### November 07, 2024
## schema

@flatfile/plugin-convert-openapi-schema@0.6.0
The release notes are as follows:

The package now uses tsup as the bundler, replacing the previous bundler. The minimum required Node.js version has been updated to 18. The test runner has been updated to Vitest, replacing Jest. Test scripts have been updated accordingly: `npm run test` to run all tests, `npm run test:unit` for unit tests, and `npm run test:e2e` for end-to-end tests. The peer dependency @flatfile/plugin-space-configure has been updated to version 0.8.0. Developers using this package should ensure they have Node.js 18 or later installed and update any scripts or tooling that reference the test commands. The package's bundle output may change due to the new bundler.
