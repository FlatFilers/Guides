### November 07, 2024
## extractors

@flatfile/plugin-foreign-db-extractor@0.4.0
The release swaps the package's bundler from Jest to Vitest. It removes the Jest configuration file and adds a new Vitest configuration file. The package.json scripts have been updated to use Vitest for running tests. There are now separate scripts for running unit tests and end-to-end tests. The Vitest configuration is imported from the @flatfile/config-vitest package.
