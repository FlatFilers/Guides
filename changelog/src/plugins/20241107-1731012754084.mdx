### November 07, 2024
## transform

@flatfile/plugin-dedupe@1.5.0
The release notes include the following updates:

The package has swapped its bundler to tsup. This is an internal change to the build process.

The package has updated its peer dependencies. The @flatfile/plugin-job-handler dependency has been updated to version 0.8.0, and the @flatfile/util-common dependency has been updated to version 1.6.0. Developers using this package should ensure they have compatible versions of these dependencies installed.

The package has increased its minimum required Node.js version to 18. Developers using this package must now have Node.js version 18 or higher installed.

The package has migrated from Jest to Vitest for testing. This is an internal change to the testing framework and should not affect the package's functionality.

The package has removed the jest.config.cjs file, as it is no longer needed with the switch to Vitest.
