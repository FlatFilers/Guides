### November 07, 2024
## transform

@flatfile/plugin-graphql-schema@1.5.0
The release introduces the following updates to the @flatfile/plugin-graphql-schema package:

The package now uses tsup as its bundler. This swaps out the previous bundler for tsup.

The minimum supported Node.js version has been updated to >=18.0.0.

The @flatfile/plugin-space-configure dependency has been updated to version 0.8.0.

The test setup has been updated to use Vitest instead of Jest. The commands for running tests have been updated accordingly: `npm run test` runs all tests, `npm run test:unit` runs unit tests, and `npm run test:e2e` runs end-to-end tests.

The @flatfile/config-vitest package has been added as a development dependency to configure Vitest.
