### November 22, 2024
## extractors

@flatfile/plugin-json-extractor@0.11.0
This release adds support for parsing JSONL (JSON Lines) files in the JSON Extractor plugin. The plugin now accepts files with the .jsonl or .jsonlines extension and treats them as a sequence of JSON objects, one per line. Empty lines are skipped, and invalid lines are ignored with a console warning during parsing. The existing support for parsing regular JSON files remains unchanged.
