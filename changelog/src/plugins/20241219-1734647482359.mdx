## transform

@flatfile/plugin-graphql-schema@1.5.1
This release updates the @flatfile/plugin-graphql-schema package to version 1.5.1. The update includes a patch change that adds a type for the tick() function used in the callback function of the configureSpaceGraphQL function. Additionally, it updates the dependency @flatfile/plugin-space-configure to version 0.8.1. For developers using this package, they should be aware of the new type for the tick() function parameter in the configureSpaceGraphQL callback function. The tick() function is now typed as TickFunction, which may require updates to existing code that uses this function.
