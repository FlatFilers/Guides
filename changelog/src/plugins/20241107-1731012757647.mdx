### November 07, 2024
## extractors

@flatfile/plugin-delimiter-extractor@2.5.0
The delimiter extractor plugin has been updated with the following changes:

The package bundler has been swapped to tsup. The @flatfile/util-extractor dependency has been updated to version 2.3.0. The Jest testing framework has been replaced with Vitest. The minimum required Node.js version has been increased to 18. The build scripts have been updated to use the new testing framework. The jest.config.cjs configuration file has been removed and replaced with a vitest.config.ts file. The package version has been bumped to 2.5.0.
