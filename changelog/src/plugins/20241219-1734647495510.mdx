### December 19, 2024
## extractors

@flatfile/plugin-zip-extractor@0.8.1
This release updates the @flatfile/plugin-zip-extractor package to version 0.8.1. It adds a type definition for the tick() function used in the job handler. The tick() function is used to report progress during the job execution. The release also updates the dependency @flatfile/plugin-job-handler to version 0.8.1.
