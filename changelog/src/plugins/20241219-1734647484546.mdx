## schema

@flatfile/plugin-convert-json-schema@0.7.1
This release updates the @flatfile/plugin-convert-json-schema package to version 0.7.1. The updates include adding a type for the tick() function and updating the dependency @flatfile/plugin-space-configure to version 0.8.1. The tick() function is used for reporting progress and messages during job execution, and the new type definition will provide better type safety when using this function. Developers using this package should update their dependencies accordingly.
