The generate blueprint task has been updated to remove the retryContext parameter from the modelParams object when using a ChatGPT model. Additionally, when using a ChatGPT model, the modelName parameter is set to the portion of the model string following the 'chatgpt/' prefix. For example, if the model is 'chatgpt/gpt-4', the modelName parameter will be set to 'gpt-4'.
