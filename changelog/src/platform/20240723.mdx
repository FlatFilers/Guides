        `@flatfile/api@1.9.7`

        This release updates the `@flatfile/api` package from version 1.9.6 to 1.9.7. The major changes are:

1. Added new filter parameters for the `listWorkbooks` function. Developers can now filter workbooks by name, namespace, label, and treatment using the new `name`, `namespace`, `label`, and `treatment` parameters respectively.

2. Added a new `includeSheets` parameter to the `listWorkbooks` function. When set to true, it will include the associated sheets for each returned workbook.

For example, to list all workbooks with a specific name and include their sheets:

```js
const workbooks = await client.workbooks.listWorkbooks({
  name: 'my-workbook', 
  includeSheets: true
});
```

3. Updated internal versioning headers sent with API requests.
