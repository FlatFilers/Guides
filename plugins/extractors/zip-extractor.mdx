### Overview

This plugin parses all Sheets in an ZIP file and extracts them into Flatfile.

```SSH Install
npm i @flatfile/plugin-zip-extractor
```

---

## Import

```js
import { zipExtractorPlugin } from "@flatfile/plugin-zip-extractor";
```

## Usage

Listen for an ZIP file to be uploaded to Flatfile. The platform will then extract the file automatically. Once complete, the file will be ready for import in the Files area.

```js
listener.use(zipExtractorPlugin());
```
