## Parameters

<ParamField path="sheetId" type="string" required>
  The `sheetId` parameter specifies the ID of the sheet you wish to monitor.
</ParamField>

<ParamField path="options.chunkSize" default="3000" type="number" optional>
  The `chunkSize` parameter allows you to specify the quantity of records to in
  each chunk.
</ParamField>

<ParamField path="options.parallel" default="1" type="number" optional>
  The `parallel` parameter allows you to specify the number of chunks to process
  in parallel.
</ParamField>

## Usage

```SSH install
npm i @flatfile/plugin-autocast
```

```js import
import { autocast } from "@flatfile/plugin-autocast";
```

```js
listener.use(autocast("us_sp_123456789"));
```
