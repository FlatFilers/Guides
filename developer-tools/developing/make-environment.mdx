---
title: "Create an environment"
api: "POST https://platform.flatfile.com/api/v1/environments"
---

This endpoint creates a new environment. `name` and `isProd` are required to create an environment.

<Warning>You'll first need to retrieve an [Account level bearer token](../security/account-token). </Warning>

### Body

<ParamField
  body="name"
  type="string"
  required
  placeholder="isolated_dev_myname"
  default="isolated_dev_myname"
>
  The name of the environment
</ParamField>

<ParamField body="isProd" type="boolean" required default="false" placeholder="false">
  Whether or not the environment is a production environment
</ParamField>
