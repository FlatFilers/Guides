Flatfile is an event-driven platform. It was designed to be highly flexible and customizable.



## Listener

The Flatfile Listener is a core building block of the Flatfile platform. 
Listening and responding to events is what enables developers to build powerful integrations with Flatfile.

## CLI

The Flatfile CLI is a command-line tool that enables developers to build reliable complex listeners with ease.
