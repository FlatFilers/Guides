## <Icon icon="satellite-dish" size={24} /> Listener

The Flatfile Listener is a core building block of the Flatfile platform. It allows developers to listen and respond to events, enabling the creation of powerful integrations. This package manages all events in your data pipeline, from data arrival to final transformation, acting as your event command center.

## <Icon icon="terminal" size={24} /> CLI

The Flatfile CLI is a command-line tool that simplifies the development and deployment of listeners. It provides commands to run local listeners, deploy agents, and manage existing deployments, making it easier for developers to build reliable and complex integrations with Flatfile.
