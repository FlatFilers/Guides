An intro about generating backups and why

## Creating Backups

Once the diff view is available, developers will be given the option to "trigger" a diff view to appear when bulk transforms occur....sort of like a "You just performed a bulk action....do you accept the updates?"
This would be something that would probably be a more robust way of using the snapshot capabilty
So in my mind, it sort of feels like a feature that should have it's own dedicated page. As there are really 2 big use cases for someone using flatfile:
Creating backups or snapshots
Creating diffs (which uses snapshots under the hood)
