---
title: "Guest Sidebar"
description: "Learn how to limit what Guests see in their Sidebar."
---

<Info>This configuration only impacts Guests, not Admins. [Learn more about user access](../security/roles-and-permissions).</Info>

Flatfile allows you to update your sidebar to hide/show certain elements via the <Tooltip tip="Learn more about Spaces">[Space](../concepts/spaces)</Tooltip> endpoint.

Simply update `metadata.sidebarConfig.` when calling `spaces.update()`.

---

<RequestExample>
```ts
const updateSpace = await flatfile.spaces.update(spaceId, {
    environmentId: environmentId,
    metadata: {
      sidebarConfig: {
        //See reference for all possible variables
      }
    }
  });
```
</RequestExample>

### `metadata.sidebarConfig`
![Sidebar Config](../images/customize/sidebarConfig.png)

<ParamField path="showGuestInvite" type="boolean"> The Collaborators link in the sidebar allows anyone to add another Guest to the Space.</ParamField>
<ParamField path="showDataChecklist" type="boolean">The data checklist shows all of the data needed for uploading to the primary Workbook. </ParamField>
<ParamField path="showSidebar" type="boolean">The sidebar as a whole can be hidden. </ParamField>
